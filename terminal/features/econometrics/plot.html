plot

usage: plot
            [-c {OPTIONS}]
            [-h] [--export {png,jpg,pdf,svg}]

Plot data based on the index. This functionality does not work for Panel Data unless creating a subset of the data that has only a singular index.

optional arguments:
  -c {OPTIONS} --column {OPTIONS}
                        Column to plot along the index (default: None)
  -h, --help            show this help message (default: False)
  --export {png,jpg,pdf,svg}
                        Export figure into png, jpg, pdf, svg (default: )
