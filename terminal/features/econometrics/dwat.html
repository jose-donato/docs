Econometrics

dwat​

Description:​

Show autocorrelation tests from Durbin-Watson. Needs OLS to be run in advance with independent and dependent variables

Usage:​

usage: dwat [-p]

Parameters​

NameDescriptionDefaultOptionalChoices
plotPlot the residualsNoneFalseNone
