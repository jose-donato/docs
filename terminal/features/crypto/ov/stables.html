stables

usage: stables [-l N] [-s {Rank,Name,Symbol,Price,Change_24h,Exchanges,Market_Cap,Change_30d}] [--export {csv,json,xlsx}] [-h]

Shows stablecoins by market capitalization. Stablecoins are cryptocurrencies that attempt to peg their market value to some external reference like the U.S. dollar or to a commodity's price such as gold. You can display only N number of coins with --limit parameter. You can sort data by Rank, Name, Symbol, Price, Change_24h, Exchanges, Market_Cap, Change_30d with --sort.

optional arguments:
  -l N, --limit N     display N number records (default: 15)
  -s {Rank,Name,Symbol,Price,Change_24h,Exchanges,Market_Cap,Change_30d}, --sort {Rank,Name,Symbol,Price,Change_24h,Exchanges,Market_Cap,Change_30d}
                        Sort by given column. Default: Rank (default: Rank)
  --pie                 displays market cap distribution across stablecoins
  --export {csv,json,xlsx}
                        Export dataframe data to csv,json,xlsx file (default: )
  -h, --help            show this help message (default: False)

Example:

2022 Feb 15, 08:16 (✨) /crypto/ov/ $ stables

First 15 stablecoins have a total 182.582 B dollars of market cap.
