# asc

Print up to 25 small cap stocks with earnings growth rates better than 25%. [Source: Yahoo Finance]

### Usage

```python
usage: asc [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display. | 5 | True | None |
