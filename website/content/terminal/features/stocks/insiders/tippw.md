# tippw

Print top insider purchases of the week. [Source: OpenInsider]

### Usage

```python
usage: tippw [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of datarows to display | 10 | True | None |
---
