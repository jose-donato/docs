# blis

Print latest insider sales 100k. [Source: OpenInsider]

### Usage

```python
usage: blis [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of datarows to display | 10 | True | None |
