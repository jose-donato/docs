# select

Select stock ticker and alter the industry, sector, country and market cap accordingly for this ticker.

### Usage

```python
usage: select -t TICKER
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| ticker | Stock ticker | None | False | None |
