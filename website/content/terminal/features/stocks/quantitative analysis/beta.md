# beta

Displays a scatter plot demonstrating the beta of two stocks or ETFs.

### Usage

```python
usage: beta [-r REF]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| ref | Reference ticker used for beta calculation. | SPY | True | None |
