# ath

All time high data for loaded coin

### Usage

```python
usage: ath [--vs {usd,btc}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| vs | currency | usd | True | usd, btc |
