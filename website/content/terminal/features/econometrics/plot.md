# plot

Plot data based on the index

### Usage

```python
usage: plot [-v VALUES]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| values | Dataset.column values to be displayed in a plot | None | True | None |
