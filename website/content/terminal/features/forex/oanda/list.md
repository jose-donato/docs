# list

List order history

### Usage

```python
usage: list [-s STATE] [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| state | List orders that have a specific state. | ALL | True | None |
| limit | Limit the number of orders to retrieve. | 20 | True | None |
