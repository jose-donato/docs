# quote

Get stock quote

### Usage

```python
usage: quote [-t TICKER]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| ticker | Ticker to get quote for. Can be in form of 'tick1,tick2...' | None | True | None |
