# distr

Compute distribution of daily returns

### Usage

```python
usage: distr [-p {mtd,qtd,ytd,3m,6m,1y,3y,5y,10y,all}]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| period | The file to be loaded | all | True | mtd, qtd, ytd, 3m, 6m, 1y, 3y, 5y, 10y, all |
