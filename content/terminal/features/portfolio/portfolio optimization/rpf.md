# rpf

Remove one of the portfolios

### Usage

```python
usage: rpf [-pf PORTFOLIOS]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| portfolios | portfolios to be removed from the saved portfolios |  | True | None |
