# history

Display account history

### Usage

```python
usage: history [-a ACCOUNT] [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| account | Symbol of coin of account or id | BTC | True | None |
| limit | Limit parameter. | 20 | True | None |
