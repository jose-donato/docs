# calendar

Show Calendar Data

### Usage

```python
usage: calendar [-d DAYS]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| days | The number of days to search for, up to 30 forward or backward use negative numbers to search back. | 7 | True | None |
