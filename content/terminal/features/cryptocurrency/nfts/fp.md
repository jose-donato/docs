# fp

Display floor price of a certain NFT collection. [Source: https://nftpricefloor.com/]

### Usage

```python
usage: fp -s SLUG
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| slug | NFT floor price collection slug (e.g., bored-ape-yacht-club) | None | False | None |
