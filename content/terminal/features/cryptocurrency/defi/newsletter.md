# newsletter

Display DeFi related substack newsletters. [Source: substack.com]

### Usage

```python
usage: newsletter [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Number of records to display | 10 | True | None |
