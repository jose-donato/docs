# global

Shows global statistics about Crypto Market

### Usage

```python
usage: global [--pie]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| pie | Flag to show pie chart with market cap distribution. Works only with CoinGecko source | False | True | None |
