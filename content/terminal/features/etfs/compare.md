# compare

Compare selected ETFs [Source: StockAnalysis]

### Usage

```python
usage: compare -e NAMES
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| names | Symbols to compare | None | False | None |
