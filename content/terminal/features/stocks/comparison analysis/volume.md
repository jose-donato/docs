# volume

Historical volume comparison between similar companies.

### Usage

```python
usage: volume [-s START]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| start | The starting date (format YYYY-MM-DD) of the stock | 2021-11-17 | True | None |
