# toppm

Print top officer purchases of the month. [Source: OpenInsider]

### Usage

```python
usage: toppm [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of datarows to display | 10 | True | None |
