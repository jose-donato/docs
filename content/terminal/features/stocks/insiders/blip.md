# blip

Print latest insider purchases 25k. [Source: OpenInsider]

### Usage

```python
usage: blip [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of datarows to display | 10 | True | None |
