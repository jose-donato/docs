# tispw

Print top insider sales of the week. [Source: OpenInsider]

### Usage

```python
usage: tispw [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of datarows to display | 10 | True | None |
