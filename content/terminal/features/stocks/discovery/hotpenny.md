# hotpenny

Provides top penny stocks from various websites. [Source: Yfinance]

### Usage

```python
usage: hotpenny [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | limit of stocks to display | 5 | True | None |
