# active

Print up to 25 top most actively traded intraday tickers. [Source: Yahoo Finance]

### Usage

```python
usage: active [-l LIMIT]
```

---

## Parameters

| Name | Description | Default | Optional | Choices |
| ---- | ----------- | ------- | -------- | ------- |
| limit | Limit of stocks to display. | 5 | True | None |
---
